Wii Balance Board for the Web

A Wii Balance Board implementation using WebHID API by @KevinPicchi and @ConcreteSciFi

Check the source code on Github

Step 1

Enable WebHID (available from chrome 85 by enabling the flag #enable-experimental-web-platform-features at chrome://flags)

Then pair your Wii Balance Board on your computer using Bluetooth (tap the sync button that is hidden next to the battery)

Click button bellow and have fun!
